0.Вход на сайт перед каждым тестом
Ожидаемый результат: вход на сайт и вход на страницу с логином и паролем
Шаги воспроизведения:
 1)открыть сайт https://www.saucedemo.com
 2)найти поле "username", ввести логин standard_user
 3)найти поле "password", ввести пароль secret_sauce
 4)найти и нажать кнопку "Login"
Фактический результат: пользователь попадает на главную страницу сайта

1. Добавление товара в корзину через вкладку товара
Ожидаемый результат: пользователь нажимает на товар, открывается страница с товаром, пользователь нажимает на кнопку "add to cart" и добавляет товар в корзину
Шаги воспроизведения:
 1)найти 4 элемент с товаром на странице
 2)найти в блоке заголовок с названием товара
 3)кликнуть на заголовок
 4)на странице товара найти блок с описанием товара
 5)в блоке найти кнопку "add to cart", кликнуть
Фактический результат: Пользователь добавляет товар в корзину из вкладки товара

2. Возвращение из корзины на вкладку товаров
Ожидаемый результат: пользователь добавляет 1 товар на странице в корзину, переходит в корзину и возвращается обратно на вкладку товаров
Шаги воспроизведения:
 1)найти 1 элемент с товаром на странице
 2)в блоке этого товара найти кнопку "add to cart", кликнуть
 3)найти на странице икноку корзины, кликнуть
 4)на странице найти кнопку "continue shopping", кликнуть
Фактический результат: пользователь после добавления товара в корзину переходит на страницу корзрины и возвращается на главную страницу

3. Фильтрация по цене max
Ожидаемый результат: пользователь выбирает фильтрацию товаров по максимальной цене
Шаги воспроизведения:
 1)найти на странице раскрывающееся меню с иконкой фильтра, кликнуть
 2)в меню выбрать 4 фильтр "Price (high to low)", кликнуть
 3)Найти 1 элемент на странице
 4)Проверить что у этого элемента на странице заголовок "Sauce Labs Fleece Jacket"
Фактический результат: после выбора фильтра пользователь видит товары с максимальной ценой

4.Выход из аккаунта
Ожидаемый результат: пользователь выходит из аккаунта
Шаги воспроизведения:
 1)найти на странице "бургер"-меню (три горизонтальные полосы), кликнуть
 2)найти 3 элемент "Logout", кликнуть
 3)проверить, что на странице есть кнопка авторизации "Login"
Фактический результат: пользователь выходит из аккаунта и возвращается на страницу входа

5. Удаление товара из корзины
Ожидаемый результат: пользователь добавляет товар в корзину, переходит в корзину и удаляет товар из корзины
Шаги воспроизведения:
 1)найти 1 элемент с товаром на странице
 2)в блоке этого товара найти кнопку "add to cart", кликнуть
 3)найти на странице икноку корзины, кликнуть
 4)на странице найти кнопку "Remove", кликнуть
Фактический результат: пользователь удаляет товар из корзины

6.Покупка 2 товаров
Ожидаемый результат: пользователь добавляет 2 товара в корзину, вводит данные для заказа и оформляется заказ
 1)найти 1 элемент с товаром на странице
 2)в блоке этого товара найти кнопку "add to cart", кликнуть
 3)найти 2 элемент с товаром на странице
 4)в блоке этого товара найти кнопку "add to cart", кликнуть
 5)найти на странице икноку корзины, кликнуть
 6)на странице найти кнопку "Checkout", кликнуть
 7)заполнить поля в форме - First Name, Last Name, Zip/Postal Code
 8)на странице найти кнопку "Continue", кликнуть
 9)проверить что на странице отображается 2 товара
 10)на странице найти кнопку "Finish", кликнуть
 11)проверить что заказ овормлен
 12)на странице найти кнопку "Back Home", кликнуть
Фактический результат: полльзователь добавляет 2 товара в корзину, переходит, оформляет заказ, после успешного оформления возвращается на главную страницу
